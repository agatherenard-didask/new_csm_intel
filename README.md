# CSM Dashboard

Dashboard de suivi des comptes clients pour les équipes Customer Success.
Permet de visualiser l'état de santé des comptes, les scores de risque et les actions prioritaires.
Les données sont définies dans `data.js`, les règles de scoring dans `score.js`, et la logique d'affichage dans `app.js`.
Aucune dépendance serveur : le projet tourne entièrement en local dans un navigateur.
